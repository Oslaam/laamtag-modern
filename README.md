# LaamTag Hub: Next-Gen Web3 Identity

**Live Application:** [app.uselaamtag.xyz](https://app.uselaamtag.xyz)  
**Target Platform:** Solana Mobile (Seeker)

## 🌟 Overview
LaamTag Hub is a social identity ecosystem built for the **Solana Mobile Phone users(SEEKER) Exclusively**. It provides a unified portal for users to manage their on-chain presence through a tiered reward system and dual-token economy.

### 🪙 Token Economy & Rewards
- **TAG (TAG):** Utility currency for games, ecosystem interactions, and "Zip" logic entry fees.
- **LAAM (LAAM):** Identity and governance token that determines user "Tier" levels.
- **SEEKER ($SKR):** Primary ecosystem token for payments and rewards.
- **SOLANA ($SOL) & USDC ($USDC):** Distributed as performance rewards in the PvP Arena and "Zip" games.

## 📱 Solana Mobile Integration
LaamTag Hub is optimized for the **Solana Mobile Stack (SMS)** to provide a hardware-secured, mobile-first experience.

- **SeekerGuard & Seed Vault:** Full integration with the Seeker’s hardware-level security for transaction signing and private key protection.
- **Mobile Wallet Adapter (MWA):** Uses the MWA protocol to connect seamlessly with mobile wallets without leaving the app.
- **Seeker ID Support:** HUMAN-readable identity integration using the `.skr` domain naming system.

### 🔌 Supported Wallets
Users can connect and sign transactions using:
* **Seed Vault Wallet** (Native Seeker Wallet)
* **Phantom Mobile**
* **Solflare**
* **Backpack**

## 🛠 Tech Stack
- **Frontend/Backend:** React & Next.js (App Router)
- **Blockchain:** Solana (Anchor Framework / Metaplex)
- **Database:** Prisma ORM
- **Targeting:** Solana Mobile Stack (SMS) / Seeker Store 2.0

## 🚀 Key Features
- **Modern Minting:** Seamless interface for users to mint identity badges directly on-chain.
- **PvP Arena:** Tier-based competition with an AI-integrated coaching system.
- **Zip Game Logic:** Level-based difficulty games with randomized chest rewards.
